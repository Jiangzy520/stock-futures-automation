Open Source

A 股实时扫描与信号看板

开源版保留了实时扫描、信号看板、推送图片和交易日定时启动相关代码， 去掉了运行期数据、私有 API、个人联系方式和部署环境里的敏感配置。

这个页面更适合作为一个可二次开发的演示模板，适合做盘中信号验证、 自建策略面板和服务器常驻扫描入口。

核心能力 多 API 分配 / 实时信号 / Web 看板 / 定时调度
适用场景 研究验证 / 自建看板 / 服务器常驻扫描
当前仓库默认不分发 `.guanlan` 运行目录；首次启动前请自行准备 watchlist、 token、运行日志目录和 AI 配置。

开源版包含内容

1. 实时扫描与信号写出

保留多 API 分配、实时扫描、信号落盘和推送图片生成逻辑，适合继续做盘中选股实验。

2. Web 看板模板

保留 `webapp` 页面和接口，便于直接接上自己的 watchlist、assignment 和运行日志。

3. systemd 与启动脚本示例

附带定时启动、收盘停止和一键拉起脚本，适合部署到 Linux 服务器持续运行。

仅供研究和交流参考，不构成任何投资建议，股市有风险。

基础股票池过滤（硬性条件）

1. 业绩预告预亏剔除

若标的发布最新业绩预告，且预告类型为“预亏”（首亏、续亏、增亏），则当日及预告后首个交易日剔除。

2. 每股收益（EPS）为负剔除

根据最新已披露的定期报告，若基本每股收益小于 0，则永久剔除，直至后续财报重新转正。

3. 动态市盈率（PE TTM）为负剔除

根据最新财报计算的滚动市盈率，若 PE（动）小于 0，则永久剔除；通常由负 EPS 引起，这里相当于再做一层保险。

日内分时形态识别逻辑

策略通过识别分时图上的关键结构点位，在盘中持续跟踪右侧确认动作。 当前核心识别点包括 R1、R2、L1、L2， 当价格向上突破 L2 时发出买入信号。 这部分逻辑已经接进实时扫描流程，用来从全市场筛选后的股票池里自动找机会。
