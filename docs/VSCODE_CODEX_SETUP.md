# 环境准备：下载 VS Code 并安装 Codex 扩展

为了保证后续开发、调试和扩展安装流程统一，建议先通过官方渠道安装 Visual Studio Code，再在扩展市场中完成 Codex 扩展的安装。

## 1. 下载并安装 Visual Studio Code

请先访问 Visual Studio Code 官方下载页面：

- https://code.visualstudio.com/download

根据你当前使用的操作系统选择对应安装包：
- Windows 用户建议优先选择 `User Installer`
- macOS 用户根据芯片类型选择 `Intel` 或 `Apple silicon`
- Linux 用户根据发行版选择 `.deb`、`.rpm` 或对应的安装方式

Visual Studio Code 是一个轻量、稳定并且扩展生态完善的开发环境，适合作为后续编写代码、查看项目结构、安装插件和执行命令的统一工作台。

安装完成后，建议先正常启动一次 VS Code，确认编辑器可以打开并进入主界面。

## 2. 在扩展市场中安装 Codex 扩展

启动 VS Code 后，在左侧活动栏打开 `Extensions` 扩展视图，你也可以直接使用快捷键：

- Windows / Linux：`Ctrl + Shift + X`
- macOS：`Shift + Command + X`

进入扩展市场后，在搜索框中输入：

- `Codex`

然后在搜索结果中找到 Codex 扩展并点击 `Install` 完成安装。

安装完成后，扩展会出现在已安装扩展列表中。后续你就可以继续进行登录、授权、打开项目和实际协作开发等步骤。

## 4. 使用建议

- 建议始终从官方渠道下载 VS Code，避免使用来源不明的安装包
- 建议优先通过 VS Code 内置扩展市场安装扩展，便于后续升级和维护
- 如果处于公司网络或受限网络环境，建议提前确认 VS Code 扩展市场可以正常访问
