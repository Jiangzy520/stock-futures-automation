# 论坛发布说明

这份公开仓库适合配合论坛帖子一起发布。

## 仓库定位

建议在论坛里把这个仓库描述成：

`一个用于展示量化网站前端壳子和系统结构的公开版仓库。`

不要把它描述成：
- 完整策略开源
- 线上生产系统全量代码
- 可直接复制运行的真实交易系统

## 推荐发帖时的说明句

可以直接用下面这段：

```text
GitHub 里放的是我整理过的公开版网站壳子，主要保留了页面结构、运行入口和公开演示接口。
真实策略、私有数据源接入、服务端扫描逻辑和线上配置都已经移除，仓库更适合用来展示系统架构和产品形态。
```

## 推荐在帖子中强调的点

- 这是公开版，不包含生产策略
- 重点是展示网站产品形态和系统思路
- 真实数据源、策略和执行逻辑已做脱敏处理
- 可以本地运行一个最小演示页面

## 推荐配合仓库一起放的内容

- 论坛主帖链接
- 首页截图
- 三模块说明图
- 一张架构图

## 不建议在论坛中写入仓库的内容

- 真实策略名称
- 真实策略参数
- 私有 API 地址
- Token、Webhook、密码、IP
- 线上服务名和部署细节
