# 股票、期货模拟盘账号与接口说明

这份文档重点说明公开可分享的模拟盘账号申请方式、接口接入路径和测试环境准备方式，不展开私有策略、内部参数或生产环境配置。

## 一、文档定位

这份说明主要回答三件事：
- 期货模拟盘使用什么接口与账号体系
- 股票模拟盘使用什么接口与测试账号体系
- 官方测试账号应当如何申请和接入

## 二、期货模拟盘：天勤量化 / 快期模拟

### 1. 接口与账号定位

期货侧更适合写成：

`使用天勤量化（TqSdk）作为期货量化接入接口，使用信易账户/快期模拟账号完成模拟环境登录与验证。`

从官方产品页可以确认：
- 天勤量化提供 Python API
- 支持回测、模拟和实盘一体化
- 免费版支持模拟交易
- 专业版和企业版提供更完整的接入能力与试用入口

### 2. 如果只是申请期货模拟账号

根据信易科技官方模拟账号注册说明，流程可以整理为：

1. 打开信易科技官方模拟账号注册说明页  
   官方说明页：https://www.shinnytech.com/register-intro/

2. 在官方说明页中点击“注册模拟账号”入口，并选择“免费注册”

3. 根据页面要求填写注册信息  
   官方说明明确提到，注册完成后需要完成邮箱或手机验证。

4. 完成激活  
   激活链接会通过注册邮箱或手机短信发送。

5. 激活成功后，注册时使用的邮箱地址、手机号或用户名，加上对应密码，即可作为“快期模拟”账号使用

6. 该账号可直接用于：
- 天勤量化（TqSdk）
- 快期 V3 Pro
- 快期 APP
- 用户中心

### 3. 如果需要更完整的试用权限

根据信易科技官方天勤量化产品页：
- 专业版提供股票实时行情、ETF 期权实时行情、股票模拟交易功能等能力
- 专业版和企业版页面都提供“申请试用”入口

因此，如果你的目标不只是基础期货模拟，而是：
- 获取更完整行情能力
- 使用股票相关模拟能力
- 使用专业版或企业版接口能力

更适合通过官方产品页申请试用：
- 天勤量化产品页：https://www.shinnytech.com/products/tqsdk

### 4. 工程侧建议写法

如果要在 GitHub 或产品说明里用更专业的方式表达，可以写成：

`期货模拟盘侧采用天勤量化（TqSdk）作为主要接入接口。基础模拟测试可通过信易账户注册“快期模拟”账号完成；如需更完整的行情、股票模拟或专业版能力，可通过天勤量化官方产品页申请试用。`

## 三、股票模拟盘：中泰 XTP 测试账号

### 1. 接口与账号定位

股票侧更适合写成：

`使用中泰 XTP 作为股票量化接口，通过 XTP 官网申请官方测试账号，在测试环境中完成 API 联调、客户端验证和模拟盘接入。`

### 2. 官方测试账号申请流程

根据中泰 XTP 官方《测试账号申请指引》PDF，普通账户测试账号申请流程可以整理为：

1. 打开 XTP 官网  
   官网：https://xtp.zts.com.cn

2. 点击“用户登录”  
   如果是首次使用，需要先在登录页选择“注册新用户”。

3. 按要求完成注册  
   官方文档特别提醒，注册信息应准确填写，因为后续测试账号及相关信息会发送到注册邮箱或手机。

4. 完成账号激活  
   XTP 会通过注册邮箱或手机短信发送激活链接。激活后，即可用注册时填写的用户名（手机号）和密码登录 XTP 官网。

5. 登录官网后提交测试账号申请  
   官方文档说明，用户登录后可根据自身需求申请 XTP 测试账号。

6. 测试账号类型  
   官方文档说明，每个 XTP 官网用户可申请：
- 一个股票类型测试账号
- 一个算法类型测试账号

7. 等待邮箱回执  
   提交申请后，测试账号会发送到注册邮箱。

8. 测试账号使用范围  
   官方文档说明，邮件回复的测试账号和密码不可修改，并且属于 XTP 平台通用测试账号，可同时用于：
- 测试 API
- 测试交易客户端

### 3. API 和客户端测试如何准备

根据官方测试账号申请指引：

- 测试 XTP-API 时  
  可在“文档中心”下载行情 API 文档、交易 API 文档；  
  可在“产品中心 -> XTP 下载”下载 SDK；  
  可在“文档中心 -> 技术文档”查看 API 使用指引和常见问题。

- 测试手工交易客户端时  
  可在“产品中心 -> XTP 下载”下载对应测试客户端。

- SmartX 客户端  
  可直接使用官网申请到的 XTP 通用测试账号进行测试。

- 迅投 QMT 等其他客户端  
  官方文档说明需要联系客户经理申请测试账号。

- 如果需要额外权限  
  如 SmartX 的 CSV 下单、算法 CSV、Python 组件、功夫模块等权限，官方文档说明可联系客户经理，由客户经理联系 XTP 运营人员或 PB 服务支持开通。

### 4. 工程侧建议写法

如果要在 GitHub 或产品说明里用更专业的方式表达，可以写成：

`股票模拟盘侧采用中泰 XTP 作为主要证券量化接口。测试环境账号通过 XTP 官网注册并申请，测试账号可用于 API 联调与客户端验证；如需算法账号、QMT 适配或额外权限开通，则按官方流程联系客户经理处理。`

## 四、期货与股票测试账号申请的主要区别

### 1. 期货侧（天勤量化 / 快期模拟）

- 更偏“注册即可用”的模拟账号路径
- 更适合个人开发、接口验证和快速开始
- 如果只做基础模拟测试，门槛相对较低

### 2. 股票侧（XTP）

- 更偏“官网注册 + 提交测试申请”的流程
- 账号下发更正式
- 更适合 API 联调、客户端验证和券商接口测试

### 3. 一句话总结

如果只用一句话概括，可以写成：

```text
期货模拟盘侧更适合通过天勤量化和快期模拟快速完成接口测试；股票模拟盘侧更适合通过中泰 XTP 官网申请官方测试账号，完成 API 与客户端联调。
```
